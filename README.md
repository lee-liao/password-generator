# 个人密码生成器小程序

简单透明的密码生成规则 - 你可以心算出来的密码！

---

## 密码生成规则（只有你知道）

```
密码 = 关键词全拼 + 标签首字母 + 种子字符
```

### 示例

| 输入 | 值 |
|------|-----|
| 关键词 | 张三 |
| 标签 | 公司OA |
| 种子 | 520 |

**生成过程：**

```
① 关键词 "张三" 的全拼 → zhangsan
② 标签 "公司OA" 的首字母 → G S
③ 种子 "520" 的键盘映射 → 5 % 2 @ 0 )

组合：zhangsan + GS + 5%2@0)
结果：zhangsanGS5%2@0)
```

### 键盘映射规则

```
键盘第一行：
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │ 8 │ 9 │ 0 │
│ ! │ @ │ # │ $ │ % │ ^ │ & │ * │ ( │ ) │
└───┴───┴───┴───┴───┴───┴───┴───┴───┴───┘
```

种子中的每个数字会转换为：数字 + 对应符号

| 种子 | 转换结果 |
|------|----------|
| 5 | 5% |
| 520 | 5%2@0) |
| 2024 | 2@0)2@$ |

## 为什么安全？

1. **规则简单** → 你可以心算或手算出来
2. **只有你知道种子** → 别人就算知道关键词和标签也没用
3. **不用记住密码** → 只需要记住关键词、标签、种子

## 使用方法

### 输入说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 关键词 | 密码的基础，转全拼小写 | 张三、项目A |
| 标签 | 区分不同用途，取首字母大写 | 公司OA、GitHub |
| 种子 | 只有你知道的数字 | 生日、纪念日 |

### 使用流程

1. 输入关键词（如：姓名）
2. 输入标签（如：系统名称）
3. 输入种子数字（如：520）
4. 点击生成
5. 长按复制密码
6. 保存到本地（可选）

## 心算示例

假设你要为「公司OA」系统生成密码：

```
关键词：张三  → zhangsan（全拼小写）
标签：公司OA → GS（首字母大写）
种子：520    → 5%2@0)（键盘映射）

心算：
zhangsan + GS + 5%2@0)
= zhangsanGS5%2@0)
```

下次需要密码时，按同样规则算出来即可！

## 注意事项

1. 种子建议用对你有意义的数字（生日、纪念日等）
2. 关键词和标签可以公开，不用保密
3. 种子不要告诉他人
4. 定期更换重要密码
