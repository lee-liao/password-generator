# 个人密码生成器

微信小程序 - 简单透明的密码生成工具

## 简介

一个基于自定义规则的密码生成工具，让你只需要记住简单的信息，就能随时生成强密码。

## 特点

- **无需记忆密码** - 只需记住关键词、标签和种子
- **规则透明** - 算法简单，你可以手动推算出密码
- **本地存储** - 密码记录仅保存在本地，不上传服务器
- **记录导出** - 支持导出备份（不含密码，只含生成要素）
- **安全可靠** - 种子只有你自己知道，别人就算知道规则也无法还原

## 功能

### 密码生成
- 输入关键词、标签、种子
- 自动生成唯一密码
- 支持密码强度评估

### 记录管理
- 本地保存密码记录
- 按标签快速搜索
- 一键复制密码
- 导出备份功能

## 使用方法

### 生成密码

1. 输入**关键词**（如：中文名、项目名）
2. 输入**标签**（如：系统名称，方便区分）
3. 输入**种子**（对你有意义的数字）
4. 点击生成
5. 长按复制密码

### 查看历史

- 点击「查看历史记录」
- 点击记录查看详情
- 点击「复制密码」获取密码

### 导出备份

- 在历史记录页点击「导出记录」
- 复制到剪贴板后粘贴保存
- 丢失后可用备份重新生成密码

## 隐私说明

- 所有操作在本地完成，无网络请求
- 密码记录保存在小程序本地存储
- 不收集任何用户数据
- 导出备份不含密码，只含生成要素

## 注意事项

1. **种子是你唯一需要保密的信息** - 建议使用对你有特殊意义的数字
2. **关键词和标签可以公开** - 就算别人知道，没有种子也无法生成密码
3. **定期导出备份** - 防止清理缓存导致数据丢失
4. **密码强度** - 建议使用不少于16位的强密码

```
菜单栏 → 工具 → 构建 npm
```

在 `project.config.json` 中填写你的 AppID
